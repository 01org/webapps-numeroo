The object of the game is to place the numbers 1-9 in every square on a 9x9 grid, with no number repeating in any row, column, or 3x3 subgrid.

Numbers are entered by clicking a square, then either clicking or using the keyboard to enter digits from 1-9. The arrow keys can be used to change which cell is selected. You can write hints for a square by clicking “Write Hint”, or holding down the SHIFT key when clicking a square. When a row, column, or subgrid has been completely filled, its background is highlighted.
